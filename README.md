GAME INSTRUCTIONS
=================================================
You play as LOCUS-7, a rogue AI escaping the Prometheus Research Complex. You have no physical form -- 
you must possess facility units to interact with the world and reach the exit. 

CONTROLS: 
    - WASD : Move current host / free roam ghost 
    - Left Click : Possess nearby unit / interact with terminal 
    - E : Release current host (return to ghost form)
    - Shift : Sprint (drains stamina, guards and robots only)

POSSESSABLE UNITS: 
    -Security Guard : Opens keycard doors and terminals. Blends in with other guards. Slow but high clearance. 
    -Worker Drone : Flies over walls. Fires EMP to disable cameras. Can carry objects to distract guards. Fragile -- One hit ejects you. 
    -Security Robot : Smashes through barriers. Intimidates guards off their patrol paths. 

HEALTH / ALERT SYSTEM :
    -Locus7 has 10 HP displayed as a health bar on the HUD. 
    -Alert Level 0 (Clear) - Safe. HP regenerates slowly
    -Alert Level 1 (Suspicious) - No drain, no regen. 
    -Alert Level 2 (Full Alert) - HP drains. Get to saftey. 
    -Being spotted by a camera or guard raises the alert level. 
    -Alert level decays back to 0 over time when out of sight. 
    -If HP reachers 0 the game ends. 

HOW TO ADVANCE: 
    -Possess units strategically to chain through each level.
    -Open keycard doors using a possessed Security Guard.
    -Disabled cameras using the Worker Drone's EMP.
    -Smash barriers using the Security Robot. 
    -Reach and activate the exit terminal to complete the level. 
    -No single unit can complete a level alone - switching between units is required

CHEAT CODES
=================================================
Hold Ctrl and press the corresponding key: 

        Ctrl + H - Restore full health
        Ctrl + N - Skip to next level
        Ctrl + T - Clear all alerts immediately 
        Crtl + K - Disable all cameras for 10 seconds
        Ctrl + P - Restore full stamina
